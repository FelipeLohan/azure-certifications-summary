No Blob Storage, os arquivos não são colocados "soltos" na conta; eles precisam de um **Container** (semelhante a uma pasta).

- **Processo:**
    
    1. Criar o Container dentro da Storage Account.
        
    2. Definir o nível de acesso público (Privado, Blob ou Container).
        
    3. Realizar o **Upload** via Portal, Azure CLI, PowerShell ou **Azure Storage Explorer**.
        
- **Dica AZ-900:** O Azure Storage Explorer é uma ferramenta gratuita (aplicativo desktop) que facilita muito o gerenciamento de arquivos como se fosse um explorador de arquivos local.
