Diferente do Blob (acessível via URL), o **Disk Storage** é um armazenamento de bloco projetado para ser usado com **Máquinas Virtuais (VMs)**.

- **Discos Gerenciados (Managed Disks):** A Azure cuida da infraestrutura por trás do disco. Você só escolhe o tamanho e o tipo.

- **Tipos de Discos:**
    - **Ultra Disk:** Cargas de trabalho extremamente intensivas.
    - **Premium SSD:** Alta performance para produção.
    - **Standard SSD:** Cargas de trabalho leves.
    - **Standard HDD:** Backup e uso infrequente.
    
- **Snapshots:** Você pode tirar "fotos" (backups pontuais) de um disco para recuperação de desastres ou para criar novos discos a partir daquele estado.

> [!TIP] **Diferença Crucial para a Prova:**
> 
> - **Blob Storage:** Acesso via HTTP/HTTPS, para dados desestruturados.
>     
> - **Disk Storage:** Montado em VMs, para sistemas operacionais e bancos de dados.
>     
> - **Azure Files:** Compartilhamentos de arquivos gerenciados que utilizam o protocolo SMB (como um drive de rede `Z:`).
