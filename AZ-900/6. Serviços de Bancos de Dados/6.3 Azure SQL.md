É uma família de produtos baseados no motor do SQL Server. É a solução ideal para migrar aplicações que já rodam SQL Server localmente.
- **Azure SQL Database:** Banco de dados relacional inteligente e gerenciado como serviço (PaaS).
- **Azure SQL Managed Instance:** Quase 100% de compatibilidade com o SQL Server local (On-premises), ideal para "Lift and Shift" (migração rápida).
- **SQL Server em VMs:** Quando você precisa de controle total sobre o Sistema Operacional (IaaS).
