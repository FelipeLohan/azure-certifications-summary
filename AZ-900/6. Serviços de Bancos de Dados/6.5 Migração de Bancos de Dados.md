 Para mover dados de servidores locais ou de outras nuvens para o Azure, utilizamos ferramentas específicas:

- **Azure Database Migration Service (DMS):** O serviço principal que guia o processo de migração com tempo de inatividade mínimo.
- **Etapas da Migração:**
    1. **Avaliação (Assessment):** Verificar se existem incompatibilidades entre o banco de origem e o Azure.
    2. **Migração:** Transferência dos dados.
    3. **Corte (Cutover):** Alteração da aplicação para apontar para o novo banco no Azure.

**Palavras-Chave para a Prova:**
- **Cosmos DB** = NoSQL, Distribuição Global, Baixa Latência.
- **Azure SQL** = Relacional, SQL Server, PaaS.
- **MySQL/PostgreSQL** = Open Source gerenciado.
- **DMS** = Ferramenta para migração de dados.
