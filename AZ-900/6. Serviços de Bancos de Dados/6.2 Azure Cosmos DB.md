O Cosmos DB é o serviço de banco de dados NoSQL **globalmente distribuído** da Azure.

- **Características Principais:**
    - **Baixíssima latência:** Respostas em milissegundos.
    - **Escalabilidade Elástica:** Aumenta ou diminui recursos instantaneamente.
    - **Multiapapi:** Suporta APIs para MongoDB, Cassandra, Gremlin (grafos) e SQL (Core).
    
- **Criação:** Ao criar um Cosmos DB, você deve escolher a API que melhor se adapta aos seus dados e definir as regiões de replicação para garantir que o usuário acesse os dados do servidor mais próximo.
