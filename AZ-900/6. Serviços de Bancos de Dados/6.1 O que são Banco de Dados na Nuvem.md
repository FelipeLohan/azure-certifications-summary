
Existem duas categorias principais cobradas na prova:

- **Relacionais (SQL):** Dados organizados em tabelas com estruturas rígidas (linhas e colunas). Ex: SQL Server, MySQL, PostgreSQL.
    
- **Não-Relacionais (NoSQL):** Dados flexíveis, sem esquema fixo, ideais para grandes volumes de dados variados (documentos, grafos, chave-valor). Ex: Cosmos DB.
