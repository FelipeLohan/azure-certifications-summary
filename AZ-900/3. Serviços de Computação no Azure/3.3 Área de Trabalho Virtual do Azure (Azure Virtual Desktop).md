
## O que é?
Serviço de **virtualização de desktop e aplicações** hospedado na nuvem. Permite que usuários acessem Windows 10/11 e aplicativos de qualquer dispositivo, em qualquer lugar.

## O que oferece
- **Desktop completo do Windows** na nuvem (multi-sessão)
- **Aplicativos remotos** individuais (RemoteApp)
- Acesso via navegador, app ou cliente RDP
- Suporte a Windows 10/11 **multi-sessão** (exclusivo do Azure — um único SO para múltiplos usuários simultâneos)

## Casos de uso (importantes no exame)
- Trabalho remoto / home office seguro
- Funcionários temporários ou terceiros (sem dar acesso ao PC corporativo)
- Empresas com equipamentos legados (usam AVD para rodar apps modernos)
- Ambientes regulados (saúde, finanças) — dados ficam no datacenter Azure

## Benefícios
- **Segurança:** dados nunca saem do Azure — o dispositivo local só exibe a tela
- **Centralização:** gerenciar desktops em escala, sem visitar cada máquina
- **Custo:** usar Windows 10/11 multi-sessão reduz licenciamento
- **Escalabilidade:** aumentar ou reduzir usuários rapidamente
- **Licença Windows 10/11** incluída para assinantes Microsoft 365 E3+

## Responsabilidades
- **Você gerencia:** imagens de SO, aplicações, políticas, usuários
- **Azure gerencia:** infraestrutura de virtualização, rede, serviço de gateway

## Ponto-chave para o exame
> AVD é a solução Azure para **desktop como serviço (DaaS)**. O diferencial é o suporte a **Windows 10/11 multi-sessão**, exclusivo do Azure.
