
## O que é?
Catálogo online de soluções certificadas e pré-configuradas de **Microsoft e parceiros terceiros**, disponíveis para implantar no Azure.

## O que você encontra lá
- **Imagens de VMs** (Windows Server, CentOS, RHEL, Ubuntu...)
- **Aplicações SaaS** (software como serviço direto no portal)
- **Soluções de parceiros** (Fortinet, Barracuda, SAP, Oracle...)
- **Templates ARM** prontos para uso
- **Serviços gerenciados** (bancos de dados, segurança, monitoramento)
- **Ferramentas para dev** (Kubernetes, containers, CI/CD)

## Como funciona
1. Acesse o **portal Azure** → Marketplace
2. Escolha a solução desejada
3. Configure os parâmetros (região, tamanho, nome)
4. Implante diretamente na sua assinatura Azure

## Pontos importantes para o exame
- O Marketplace **não é gratuito** — soluções têm custo próprio além do custo da infraestrutura Azure
- Permite implantar soluções complexas com **poucos cliques**
- Soluções são **certificadas e validadas** pela Microsoft
- Disponível para **todos os planos** de assinatura Azure

## Exemplos de categorias
- Computação | Redes | Armazenamento | Segurança | Bancos de dados | IA + ML | DevOps
