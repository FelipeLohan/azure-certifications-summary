## Sobre o Exame AZ-900

O **AZ-900** é a certificacao de entrada da Microsoft para a plataforma Azure. Ele valida conhecimentos fundamentais sobre:

- Conceitos de nuvem (IaaS, PaaS, SaaS, tipos de nuvem)
- Servicos principais do Azure (computacao, rede, armazenamento, banco de dados)
- Seguranca, privacidade e conformidade
- Precos, SLAs e gerenciamento de custos

**Detalhes do exame:**
- Duracao: 45 minutos
- Numero de questoes: 40-60
- Pontuacao minima para aprovacao: 700/1000
- Idioma: disponivel em portugues
